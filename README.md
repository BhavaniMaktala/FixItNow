# 🔧 FixItNow

A modern and user-friendly web application that connects users with trusted home service professionals. Whether it's plumbing, electrical repairs, carpentry, painting, appliance servicing, or cleaning, FixItNow makes booking home services simple, fast, and convenient.

---

## 📖 Overview

FixItNow is designed to simplify the process of finding and booking reliable service providers for household maintenance and repair tasks. Users can browse available services, choose the required service, and schedule appointments through an intuitive and responsive interface.

The project demonstrates full-stack web development concepts, including frontend design, backend development, database integration, and user authentication.

---

- 👤 User Registration & Login

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/BhavaniMaktala/FixItNow.git
```

### Navigate to the Project

```bash
cd FixItNow
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
